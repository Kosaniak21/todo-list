## Specification:

Simple TODO-list with async react-redux

## Build Setup:

``` bash
# Download repository:
git clone https://github.com/Kosanyak19/todo-list.git

# Go to the app:
cd todo-list

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8080/
npm start

# Output will be at dist/ folder
npm run build
```
